# Media Assets

Add these files here for a stronger GitHub landing page:

- `demo.gif`: 15-30 second product demo
- `before-after.png`: one side-by-side screenshot or annotated comparison

Recommended GIF flow:

1. Focus a window on monitor 1.
2. Press `Alt+1`, `Alt+2`, `Alt+3`.
3. Press `Alt+Shift+Right`.
4. Show a different applied layout on the next monitor.

Recommended screenshot labels:

- Left: "FancyZones only"
- Right: "FancyZones + Hotkey Bridge"
